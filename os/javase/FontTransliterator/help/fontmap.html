Map characters between two fonts

Character Mapping

Select character to be mapped using the keypad. There are two keypads so characters for each font can be mapped to another set of characters.

Note: you will see the selected characters in the Mapping display

After selecting character from both the left hand side and the right hand side keypad, press 'Add' button to add this character mapping to the FontMap.

Ex:

  • I = U
  • on = OFF
  • You = Me

Note: you can add any number of character set mappings to your FontMap.

Based on the above FontMap, the line

I Love You

will be converted to

U Love Me

 

Adding Optional Rules to Character Mapping

Optionally you can add Rules to any of the FontMap Entry.

There are Four kind of rules that can be set, for the FontMap Entry.

  •  IS First Letter (in the word)
  •  IS Last Letter (in the word)
  •  IS Preceded By (symbol 1 is preceded by the selected symbol)
  •  IS Followed By (symbol 1 is followed by selected symbol)
Ex:
  • I = We           [IS First Letter]
  • You = Him     [IS Last Letter]
  • o = G             [IS Preceded By]     'L'
  • v = A             [IS Followed By]     'e'

Based on the above FontMap, the line

I Love You

will be converted to

We LGAe Him
